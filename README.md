# 申论统一修炼台

离线申论备考App，浏览器直接打开 `index.html` 即可使用，也可部署到 GitHub Pages。

## 文件结构

| 文件 | 说明 |
|---|---|
| `index.html` | 默认入口与主页面结构，适合 GitHub Pages |
| `申论统一修炼台.html` | 兼容旧入口，自动跳转到 `index.html` |
| `styles.css` | 页面样式 |
| `data.js` | 题库、卡片库、12周计划 |
| `app.js` | 页面交互、复习算法、存储逻辑 |

## 五个功能模块

| 底部Tab | 功能 |
|---------|------|
| 🏠 首页 | 今日任务看板、12周自动排程、打卡天数 |
| 📖 复习 | 翻卡间隔重复（SM-2算法）、8类133张卡片 |
| ✏️ 练习 | 得分技能精练、17技能170道微练习、30段材料阅读训练、规范词替换训练、2.5小时模考、限时倒计时、技能搜索、每日复盘单 |
| ❌ 错题 | 错题本自动收集、搜索筛选、分类重练、答对2次移出 |
| 📊 数据 | 雷达图、打卡、正确率趋势、复盘薄弱点、手册速查、设置面板 |

## 配套资料

配合「申论50→70完整备考手册」使用，手册定义了"什么时候练什么"，App提供了"怎么练怎么背"。当前复习卡片已吸收手册中的题型框架、规范词、避坑清单、资料清单和核心心法。

## 使用说明

1. 双击 `index.html` 在浏览器中打开
2. 首页自动根据当前日期计算你在第几周，推送当日学习内容
3. 按底部Tab切换功能模块
4. 所有学习记录保存在浏览器 localStorage 中
5. 练习答题会自动保存草稿，切题、返回或关闭页面后再次打开可恢复

## GitHub Pages 部署

1. 在 GitHub 新建仓库，例如 `shenlun-unified`
2. 上传本目录全部文件到仓库根目录
3. 进入仓库 `Settings → Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，Folder 选择 `/root`
6. 等待部署完成后访问 `https://你的用户名.github.io/shenlun-unified/`

> 学习记录仍保存在各设备浏览器本地，不会上传到 GitHub。跨设备使用请通过「📊 数据 → ⚙️ 设置 → 数据备份」导出/导入。

## 安全部署说明

- 本项目是纯静态前端页面，不包含服务端接口、登录系统、第三方统计脚本或 API Key。
- 页面已设置基础 Content Security Policy，限制资源只从本站加载，并禁止被其他网站嵌入。
- 数据导入会清洗备份 JSON，并限制文件大小，降低恶意备份造成页面异常或脚本注入的风险。
- 如果仓库设为公开，`data.js` 中的题库和手册内容也会公开。若内容不适合公开，请使用私有仓库或私有部署。
- 导出的备份 JSON 包含个人学习记录和答题内容，请不要提交到公开仓库。

## 数据备份

学习记录默认保存在当前浏览器中。建议定期进入「📊 数据 → ⚙️ 设置 → 数据备份」导出 JSON 备份文件，换设备或清理浏览器缓存后可通过「导入」恢复记录。
